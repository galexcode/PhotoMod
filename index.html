PhotoMod

A Facebook application made in Processing

View project onGitHub

PhotoMod is a Facebook application made in Processing where you can edit a picture and add a tint or/and overlay to it and then upload it to your pictures-folder in Facebook.

The app requires a Facebook account and should be run only on Facebook, even thought you can run it through the webserver too, but for best performance, it should be run through Facebook.

The app is a part of the Coursera course Creative Programming for Digital Media & Mobile Apps (University of London International Programmes)

You can test the app at https://apps.facebook.com/photomod/
